# 剧本评测系统升级说明

## 📋 升级概述

本次升级将剧本评测系统从原有的29维度体系重构为基于《短剧商业潜力评估报告》参考标准的19维度体系，更加聚焦商业价值和市场适配性。

## 🔄 主要变更

### 1. 维度体系重构

#### 原有体系（29维度）
- 包括黄金3秒、情绪价值密度、节奏与反转等29个维度
- 分类较为散乱，缺乏系统性
- 部分维度重叠

#### 新体系（19维度 + 4基础信息维度）

**基础信息维度**（不参与评分）：
1. AI浓度分析 - 检测AI生成内容占比
2. 收稿匹配度 - 剧本与市场需求匹配程度
3. 格式规范性 - 剧本格式符合标准
4. 制作难度 - 拍摄制作复杂度评估

**A. 市场共鸣与竞争定位**（权重30%）：
1. 目标受众定位（10%）- 痛点抓取、情感共鸣、受众画像
2. 原创性评分（8%）- 创意新颖度、差异化、创新突破
3. 当下热播契合度（12%）- 爆款对标、市场热点、算法友好

**B. 叙事与剧本基因**（权重40%）：
1. 叙事逻辑（5%）- 剧情连贯、动机合理、逻辑严密
2. 钩子强度（6%）- 开篇冲击、悬念设置、集末钩子
3. 爽点设计（6%）- 爽点密度、打脸效果、情绪峰值
4. 节奏与结构（5%）- 节奏紧凑、起承转合、结构完整
5. 主线连贯性（5%）- 主线清晰、支线关联、剧情流畅
6. 人物塑造（5%）- 人物立体、性格层次、成长弧光
7. 对话质量（4%）- 台词精彩、潜台词、金句密度
8. 悬念有效性（4%）- 悬念设置、扣人心弦、反转效果

**C. 商业化潜力**（权重18%）：
1. 用户粘性（9%）- 情绪闭环、高频打脸、极致反转
2. 传播潜力（9%）- 名场面、社交话题、金句传播

**D. 合规性评估**（权重12%）：
1. 内容合规性（6%）- 极端内容、审核风险、平台适配
2. 价值观导向（6%）- 价值正确、社会意义、正向引导

### 2. 报告格式升级

新的报告格式完全参考《短剧商业潜力评估报告》PDF格式：

```
《短剧商业潜力评估报告》
AI 智能诊断｜商业适配度·结构风险·变现潜力

I. 基础信息
   - AI浓度分析
   - 收稿匹配度
   - 格式规范性
   - 制作难度
   - 角色场景特效表

II. 总体潜力评分
   - XX/100 (X级)

III. 执行摘要
   - 频类、题材、一句话介绍
   - 剧情主线、核心结论

IV. 详细分析
   A. 市场共鸣与竞争定位
   B. 叙事与剧本基因
   C. 商业化潜力
   D. 合规性评估

V. 综合可操作建议
```

## 🚀 使用方法

### 1. 启动评测系统
```bash
python app.py
```

### 2. 访问Web界面
打开浏览器访问：http://localhost:5000

### 3. 上传剧本评测
- 选择剧本文件（.txt/.pdf/.docx）
- 选择要评测的维度（可选，默认评测所有维度）
- 点击"开始评测"

### 4. 查看评估报告
评测完成后，系统会自动生成符合PDF格式的评估报告，包含：
- 基础信息分析
- 总体潜力评分（S/A+/A/B+/B/C/D等级）
- 执行摘要
- 详细分维度分析
- 综合可操作建议

## 📊 评分等级

- **S级**（93+分）：极罕见，必火级别
- **A+级**（85-92分）：优秀，有爆火潜力
- **A级**（75-84分）：良好，具备竞争力
- **B+级**（65-74分）：中等，需优化
- **B级**（55-64分）：一般，有明显短板
- **C级**（45-54分）：较差，需大幅修改
- **D级**（45分以下）：失败，不推荐

## 🔧 技术说明

### 维度权重分布
- 市场共鸣与竞争定位：30%
- 叙事与剧本基因：40%
- 商业化潜力：18%
- 合规性评估：12%
- 基础信息：0%（仅供参考，不参与评分）

### API调用
每个维度都会调用豆包API进行评测，API返回格式为JSON，包含：
- total_score: 得分
- max_score: 满分
- grade: 等级
- analysis: 详细分析
- strengths: 优点
- improvement_areas: 可打磨点
- improvement_suggestions: 改进建议
